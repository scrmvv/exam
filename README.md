# Корочки.есть — фронтенд для экзамена

Небольшой сайт для записи на онлайн-курсы дополнительного профессионального образования.

Данные сохраняются в `localStorage` браузера, поэтому после обновления страницы заявки не пропадают.

## Доступы

Пользователь для просмотра тестовых заявок:
- логин: `user123`
- пароль: `123456`

Администратор:
- логин: `Admin`
- пароль: `KorokNET`

## Что реализовано

- регистрация пользователя с проверкой полей;
- авторизация пользователя;
- просмотр своих заявок;
- создание новой заявки на курс;
- отзыв к заявке;
- панель администратора;
- смена статуса заявки: `Новая`, `Идет обучение`, `Обучение завершено`;
- три тестовые заявки;
- слайдер изображений на главной странице;
- адаптивная верстка под телефон и компьютер;
- HTML, CSS, Tailwind CDN и JS.
